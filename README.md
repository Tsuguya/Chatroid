# Chatroid
Chatへの自動投稿お助けするWebアプリケーションです。

# インストール

## meteor のインストール

````
$ curl https://install.meteor.com/ | sh

````

## Chatroid のインストール

````
$ git clone https://github.com/Tsuguya/Chatroid.git

````

## Chatroid の起動
下記のコマンドで起動します。
ChatWorkのAPIを使うため、アクセストークンを環境変数にいれて起動してください。

````
$ CHATWORK_ACCESS_TOKEN="チャットワークのアクセストークン " meteor run

````
